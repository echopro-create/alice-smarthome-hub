# Семантическое ядро (СЯ) контентного хаба «Умный Дом»

Данный документ представляет собой семантическое ядро, разработанное по принципу **Long-tail SEO** (низкочастотные запросы с «длинным хвостом»). Ядро ориентировано на сбор поискового трафика по конкретным техническим проблемам, решениям из Telegram-чатов (Home Assistant, Aqara, Zigbee, Node-RED) и сложным сценариям автоматизации.

---

## 1. Кластер A: Базовая автоматизация и экосистема Яндекса (Алиса)
*Фокус: НЧ-запросы пользователей, расширяющих возможности Станций и базовых сценариев.*

*   **Группа A1: Сценарии по расписанию и времени**
    *   *«как настроить включение кофемашины через умную розетку яндекс при открытии двери»* (Интенсивность: средняя, высокая конверсия)
    *   *«сценарий световой будильник с плавным нарастанием яркости ламп алиса»* (Интенсивность: средняя)
    *   *«автоматическое выключение гирлянды по расписанию в доме с алисой»* (Интенсивность: низкая)
*   **Группа A2: Голосовые триггеры и кнопки управления**
    *   *«сценарий выключить весь свет одной кнопкой алиса когда уходишь»* (Интенсивность: высокая)
    *   *«как настроить беспроводную кнопку zigbee на три действия в алисе»* (Интенсивность: средняя)
    *   *«как сделать чтобы алиса приветствовала при открытии входной двери»* (Интенсивность: высокая)
