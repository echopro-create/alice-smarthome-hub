## Метод верификации количества слов
Для каждого файла будет написан текст тела, после чего будет запущен Python-скрипт для точного подсчета слов (разделение по пробельным символам) в теле файла.

## Проверка валидности Astro сборки
После добавления файлов мы запустим `npm run build` или `npx astro check` для подтверждения отсутствия синтаксических или структурных ошибок.
