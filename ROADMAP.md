# Дорожная карта развития контентного хаба «Умный Дом с Алисой»

Эта дорожная карта описывает стратегические этапы запуска, оптимизации, наполнения и монетизации портала на 2026–2027 годы.

---

## 🚀 Фаза 1: Инфраструктурный фундамент и запуск MVP (Июнь 2026)
*   **Технологический стек**:
    *   [x] Развертывание статической архитектуры на **AstroJS** (Astro v6 с использованием нового Content Layer API).
    *   [x] Разработка модульного адаптивного UI на чистом CSS (Vanilla CSS) с поддержкой CSS-переменных, адаптивной сетки Grid/Flexbox и эффектов стекломорфизма (glassmorphism).
    *   [x] Создание системы построчного хэширования файлов (`hashline_helper.py`).
*   **SEO-гигиена**:
    *   [x] Создание семантического ядра (СЯ) с упором на низкочастотные запросы с длинным хвостом (Long-tail SEO).
    *   [x] Реализация компонента генерации микроразметки **Schema.org** (`@type: "HowTo"`) в формате JSON-LD.
    *   [x] Разработка автоматизированного набора тестов (`tests/seo.test.js`) для валидации длины заголовков, мета-описаний и объема контента при каждой сборке.
*   **MVP-Контент**:
    *   [x] Создание и валидация 2 стартовых демо-материалов (утренний сценарий автоматизации и технический разбор отвала Zigbee-датчиков).

## 📈 Фаза 2: Наполнение и ускорение индексации (Июль – Август 2026)
*   **Генерация контента**:
    *   [ ] Реализация первой волны контент-плана (первые 30 статей из списка 100 запланированных).
    *   [ ] Создание уникальных скриншотов из мобильного приложения «Дом с Алисой» для каждого шага инструкций.
*   **Продвинутая SEO-интеграция**:
    *   [ ] Подключение плагина `@astrojs/sitemap` для автоматической генерации `sitemap.xml`.
    *   [ ] Написание скрипта отправки новых страниц по протоколу **IndexNow** в Яндекс и Bing для мгновенного попадания статей в топ выдачи.
    *   [ ] Добавление метатега верификации Яндекс.Вебмастера и кода отслеживания Яндекс.Метрики.
*   **Аудит производительности**:
    *   [ ] Контроль показателей Core Web Vitals на мобильных устройствах (LCP < 0.5s, CLS < 0.05, FID/INP < 50ms).
    *   [ ] Оптимизация веса изображений через современные форматы (`.webp` / `.avif`).

## 💰 Фаза 3: Монетизация и экосистемный шеринг (Сентябрь – Октябрь 2026)
*   **Яндекс Дистрибуция**:
    *   [ ] Регистрация в CPA-платформе Яндекс Дистрибуции под аккаунтом владельца.
    *   [ ] Модерация сайта и юридическое оформление (договор для самозанятого/ИП).
    *   [ ] Размещение нативных динамических виджетов Яндекс Маркета в статьях. При клике на нужный датчик или розетку пользователь видит актуальную цену и может совершить покупку, принося нам комиссионный доход.
*   **Вирусная дистрибуция сценариев**:
    *   [ ] Настройка механики генерации ссылок импорта сценариев Яндекса (`yandexShareUrl`).
    *   [ ] Оптимизация блока призыва к действию (CTA) «Добавить в Алису» для увеличения конверсии.
*   **Прямой рекламодатель**:
    *   [ ] Регистрация на WOWBlogger и аналогичных платформах.
    *   [ ] Разработка медиа-кита для публикации платных нативных обзоров новых линеек датчиков и хабов от производителей (Aqara, Tuya, Yeelight, Ясдор и др.).

## 🛠️ Фаза 4: Инструменты автоматизации и масштабирование (Конец 2026 – 2027)
*   **Интеграция Headless CMS**:
    *   [ ] Подключение Git-based CMS (например, Decap CMS) для удобного добавления материалов нанятыми копирайтерами и редакторами.
*   **Интерактивные виджеты**:
    *   [ ] Создание интерактивного «Конструктора Сценариев» на фронтенде Astro: пользователь выбирает свои датчики (например, датчик движения + умная лампа) и получает готовый пошаговый план настройки с возможностью скачать сгенерированный JSON-LD для своего личного сайта.
*   **Расширение семантики**:
    *   [ ] Сбор новой семантики по протоколам Matter и Thread, интеграция гайдов по работе с альтернативными серверами автоматизации (Home Assistant, SprutHub) в связке с Алисой.
